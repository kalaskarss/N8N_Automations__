# RAG Automation Projects (n8n + Gemini + Pinecone)

This repository showcases multiple **Retrieval-Augmented Generation (RAG)** automation workflows built using **n8n**, **Google Gemini**, and **Pinecone vector database**.

The goal of these projects is to demonstrate scalable document ingestion, semantic search, and AI-powered question answering over dynamic data sources.

## Projects Included

### RAG_1 – Google Drive FAQ Bot

Automated ingestion of documents from Google Drive and semantic search using Pinecone.

### RAG_2 – Website RAG

Scrapes website content and enables AI-based question answering.

### RAG_3 – PDF Knowledge Assistant

Allows users to upload PDFs and ask questions about their content.

### RAG_4 – Multi-Source RAG System

Combines multiple knowledge sources into a single AI-powered retrieval system.

## Tech Stack

* n8n
* Google Gemini API
* Pinecone Vector Database
* Retrieval-Augmented Generation (RAG)
* Vector Search

## Key Capabilities

* Automated document ingestion
* Semantic vector search
* AI agent-based retrieval
* Real-time knowledge updates
* Scalable RAG pipelines

These workflows demonstrate how modern AI systems integrate **LLMs with external knowledge sources** to deliver accurate and context-aware responses.
